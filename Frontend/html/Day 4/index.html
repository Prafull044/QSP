    CSS:- CSS (Cascading Style Sheets) is the foundational stylesheet language used to control the presentation, layout, and visual design of web documents.     Javascript:- JavaScript (JS) is a lightweight, interpreted, prototype-based programming language. It brings interactivity and dynamic behavior to web pages.     Task 26: HTML Attribute Purpose

    id:-A unique identifier used to target a specific HTML element with CSS or JavaScript
    class:- A non-unique identifier used to group multiple HTML elements for shared CSS styling or JavaScript actions.
    title:-Defines extra advisory information about an element, typically displayed as a tooltip on hover.
    alt:- Provides alternative descriptive text for an image if it fails to load or for screen readers.
    href:-Specifies the URL link destination for hyperlinks or linked resources.
    src:-Specifies the source path of an external media asset, such as an image or script.
    target:-Determines where to open a linked document, such as in a new browser tab
    width:- Specifies the horizontal measurement of an element in pixels or percentage.
    height:- Specifies the vertical measurement of an element in pixels or percentage.

    Task 29: Mini Website

    Welcome to A blog is an online platform

    A blog is an online platform where individuals or businesses share regularly updated information, ideas, or experiences. These posts are typically organized in reverse chronological order and can be monetized through advertising, affiliate commissions, and sponsored content.

    Technical Data Structure:-

    A blog list is a dynamic archive page. It uses a loop to pull data from your database or Content Management System (CMS).Database Query: Requests a specific number of posts (e.g., 10 per page) sorted by date.Dynamic Variables: Pulls the title, URL, image path, author ID, and timestamp for each post.Conditional Logic: Checks if a post is marked as "draft" or "published" before displaying it.

    SEO (Search Engine Optimization) Impact

    The blog list page is a major hub for search engine crawlers to find your individual articles.Internal Linking: Passes authority (Link Juice) from the main listing page down to older articles.Schema Markup: Uses CollectionPage schema to tell search engines that this page aggregates other content.Canonicals: Uses proper canonical tags to avoid duplicate content penalties when using pagination (e.g., page 2, page 3).

    Key Performance Metrics (KPIs)

    Track these data points in your analytics tool to see if your blog list is successful:Click-Through Rate (CTR): The percentage of users who click an article after viewing the list.Bounce Rate: The percentage of users who leave your site immediately after landing on the list page.Scroll Depth: How far down the list page users scroll before quitting or clicking.
